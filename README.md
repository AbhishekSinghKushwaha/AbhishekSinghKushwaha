### `$ cat about.md`

I build agentic AI systems end to end: orchestration graphs, hybrid retrieval pipelines, and the infrastructure that gets them into production. My most recent build, **VERDICT**, is a full stack contract analysis platform combining LangGraph agent DAGs, hybrid RAG (BM25 + vector + HyDE + RRF fusion + Cohere reranking), and GraphRAG on Neo4j, deployed across Render and Vercel with full LangSmith observability.

Before this I researched deep learning for biometric authentication via gait analysis at FAU (CNN/RNN pipelines on video data), and worked across full stack development and data science roles.
